# Holocron

A Star Wars encyclopedia for ships — browse and filter starships by
name, affiliation, and class, with a detail page for each ship.

Built with Next.js, TypeScript, and Tailwind CSS.

**[Live Site →](https://holocron-iota.vercel.app/)**

![Holocron screenshot](images/app/app-screenshot.png)

## Features

- Browse a catalog of Star Wars starships
- Filter by affiliation and class
- Search ships by name
- Detail page for each ship

## Tech Stack

- **Framework** — Next.js 16 (App Router)
- **Language** — TypeScript
- **Styling** — Tailwind CSS v4
- **Deployment** — Vercel

## Running Locally

```bash
git clone https://github.com/Emeyrald/Holocron.git
cd Holocron/holocron
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
